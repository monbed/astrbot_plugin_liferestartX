# 人生重开模拟器插件

适用于 AstrBot 的人生重开模拟器插件。

## 功能特点

- 随机生成人生属性和天赋
- 生成精美的图片展示
- 支持群聊开关控制

## 使用方法

### 基础指令

- `/人生重来`：开始新的人生
- `/重开`：开始新的人生（简写）
- `/重开帮助`：显示帮助信息
- `/人生重开开`：启用插件
- `/人生重开关`：禁用插件

### 游戏说明

每次重开都会随机生成：

1. 基础属性（颜值、智力、体质、家境）
2. 随机天赋
3. 人生经历
4. 人生总结
